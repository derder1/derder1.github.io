# 访问修饰符

四种访问控制修饰符号

  1. 公开级别： public，对外公开
  2. 受保护级别： protected修饰，对子类和同一个包中的类公开
  3. 默认级别：没有修饰符号，向同一个包的类公开
  4. 私有级别：private修饰，只有类本身可以访问，不对外公开
